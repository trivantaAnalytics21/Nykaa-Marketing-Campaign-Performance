# Nykaa-Marketing-Campaign-Performance

## Nykaa Campaign & Marketing Analysis using Tableau

### Project Overview

This project analyzes Nykaa marketing campaign performance, customer behavior, 
and product trends using Tableau dashboards to generate business insights and improve marketing strategies.

### Objectives

- Analyze campaign performance and ROI
- Study customer purchasing behavior
- Identify top-performing products
- Improve marketing effectiveness using insights

### Tools & Technologies

- Tableau
- Excel / CSV

### Key Features

- Interactive Tableau dashboards
- Sales and revenue trend analysis
- Customer segmentation analysis
- Campaign performance tracking
- Product category insights

### Dashboard Highlights

- Campaign Performance Overview
- Revenue and Profit Analysis
- Top-Selling Products
- Customer Purchase Trends
- Region-wise Sales Distribution
